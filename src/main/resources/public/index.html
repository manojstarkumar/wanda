Sample Usage

Send a post request to
https://wanda-avacado.herokuapp.com/rest/sendsms
with a JSON body in the following format
{ "destinationNumber": "A_MOBILE_NUMBER_WITH_COUNTRY_CODE", "textMessage": "TEXT_MESSAGE_MAX_159_CHARS" }
Also add your API Key in the header 
api-key: YOUR_API_KEY
